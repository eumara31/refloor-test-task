# Тестовое задание для Refloor
https://refloor-test.vercel.app/
## Выбор библиотеки
Для выполнения тестового задания был выбран Three, поскольку я с ним работал.
Так же рассматривал WebGL, но использование WebGL подразумевает ручную установку вершин объектов и создание освещения с помощью линейной алгебры, это бы замедлило разработку и ухудшило читаемость кода.
## Работа со светом и материалами
Все текстуры используют PBR-материалы.
Для отображения света используется два источника источник, THREE.PointLight, расположенный в центре комнаты и AmbientLight. У меня не получилось реализовать смену дневного освещения, но у PointLight присутствует регулировка яркости.
Для мягкости света используется тонмаппинг.
